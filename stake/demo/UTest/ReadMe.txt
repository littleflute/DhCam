	Demo for Multi-Camera ---- HVMViewer
	function:
		Automatically determine the current number of the working camera; 
		Automatically determine the model, and support the various types of published 1394-cameras.
		Each camera has one corresponding view window. 
		Each camera has its own corresponding dialog for adjustment parameters.

        Notes:
	To compile and link this program correctly, you need the newest 
	Microsoft Platform SDK.
		
	Version:1.0.2.9
